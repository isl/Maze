Mapping-centered overview

This functionality provides a 3D representation of source and target schema regarding mapping rules of the X3ML. The floor represents the source schema, while the target schema is depicted at the upper level.

X3ML-centered overview

As regards the X3ML mapping overview, it provides the ability to understand the structure of an X3ML file and navigate through the assertions between the source and target schema.

Instance Representation

Instance representation tries to depict the result of the mapping, providing an example. This functionality requires an export sample from the X3ML Engine.

Tracking Changes Over Time

The system provides the capability of monitoring the evolution of a mapping over time regarding the transformation rules and the evolution of the source and target schema.
