Textual Comparison

This functionality provides the simplest way to compare XML elements. Thus, the first representation of comparing two mappings files proposes a side-by-side projection comparing two X3ML files.

Graphical Comparison

Graphical representation has been a powerful technic to describe and understand data. In order to provide more sufficient functionality, this approach tries to find differences between two mappings and represent them as a 3D graph.

Instance Comparison

This functionality provides comparison about instances between different mappings by a side-by-side projection.
